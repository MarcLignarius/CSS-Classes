Classes

This particular page showcases the use of classes to style certain HTML elements. This involves both assigning different classes to the same HTML element (for instance, p class="class1" and p class="class2") as well as assigning the same class to different HTML elements (for instance, p class="class1" and li class="class1").

Other concepts covered during this project and their use can be found below:

  • The Box Model
  • Cascading CSS
  • Divs and Spans
  • Floats
  • Media Queries and Responsive Design
